# DeadSimple Claude Marketplace

Plugin marketplace for Claude Code by [DeadSimple](https://github.com/deadsimple-xyz).

## Install

```
/plugin marketplace add deadsimple-xyz/deadsimple-claude
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| [hats](https://github.com/deadsimple-xyz/hats) | BDD-driven AI dev team. Five roles: Manager, Designer, CTO, QA, Developer. |

## Install a plugin

```
/plugin install hats@deadsimple
```
